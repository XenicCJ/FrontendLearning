代码格式

code元素定义编程代码示例

var person = { firstName:"Bill", lastName:"Gates", age:50, eyeColor:"blue" }

code元素不保留多余的空格和折行

var person = { firstName:"Bill", lastName:"Gates", age:50, eyeColor:"blue" }

在pre元素中包含代码可以解决此问题

var person = {
    firstName:"Bill",
    lastName:"Gates",
    age:50,
    eyeColor:"blue"
}
